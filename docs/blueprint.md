# **App Name**: Wellbeing Compass

## Core Features:

- User Data Collection: Collect user data (name, job title, company, email, phone) and email it to rodrigo@pontosfortes.com.br along with the assessment results.
- Interactive Wellbeing Wheel: Interactive wheel chart based on Gallup's wellbeing areas, allowing users to input their current and desired scores (1-10). Display the gap between the two scores visually.
- Action Plan Definition: Enable users to select 3 areas for improvement and define 3 actionable steps with completion dates for each area. Generate a printable summary of the wheel and action plan and send it to rodrigo@pontosfortes.com.br.

## Style Guidelines:

- Primary color: A calming teal (#008080) to represent balance and wellbeing.
- Secondary color: A soft gray (#f0f0f0) for backgrounds and neutral elements.
- Accent: A vibrant orange (#FFA500) for interactive elements and highlighting areas in the wheel.
- Clean and intuitive layout with clear visual hierarchy to guide the user through the assessment process.
- Use simple, minimalist icons to represent each area of wellbeing in the wheel chart.
- Subtle animations to provide feedback when the user interacts with the wheel and defines action items.
